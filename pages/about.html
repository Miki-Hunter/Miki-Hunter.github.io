常见问题FAQ

站点数据来源是什么？Where does the data come from?
行星数据来自 NASA Solar System Exploration；深空天体数据来自 Messier Catalog、NGC 和 ESA。所有科学数据均标注来源。Planetary data from NASA Solar System Exploration; deep-sky data from the Messier Catalog, NGC, and ESA. All scientific data is sourced and dated.
为什么选择纯静态网站？Why a pure static site?
零依赖、加载快、部署简单。托管于 GitHub Pages，无需服务器成本，任何人均可 Fork 并搭建自己的天文站。Zero dependencies, fast loading, simple deployment. Hosted on GitHub Pages with no server costs — anyone can fork and build their own astronomy site.
如何参与贡献？How can I contribute?
项目完全开源。你可以在 GitHub 上提交 Issue 报告错误或提出建议，也可以 Fork 后提交 Pull Request 贡献代码或内容。The project is fully open-source. Submit Issues to report bugs or suggest ideas, or fork and send Pull Requests with code or content contributions.
太阳系模型的比例准确吗？Are the Solar System model proportions accurate?
行星大小比例和轨道周期按真实数据等比缩放。但轨道间距和行星体积因显示需要做了调整——如果完全按真实比例，内行星将几乎不可见。Planet sizes and orbital periods are scaled to real data. However, orbital spacing and planet sizes are adjusted for display — at true scale, inner planets would be nearly invisible.
可以把这个网站翻译成其他语言吗？Can I translate this site into other languages?
当然可以！欢迎 Fork 项目并创建多语言版本。我们非常乐意在友情链接中添加你的翻译站。Absolutely! Fork the project and create a multilingual version. We\'d be happy to link to your translation.
